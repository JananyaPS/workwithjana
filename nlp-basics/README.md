### **1. User Question Answering**
A foundational NLP project demonstrating how a model can answer user queries based on provided input text.

**Highlights**
- Implements basic text preprocessing (tokenization, cleaning)
- Uses embeddings or pretrained models to understand semantic meaning
- Answers user questions based on a given context passage
- Demonstrates how NLP models extract relevant information
- Helps build intuition for QA tasks used in search, chatbots, and conversational AI

- Understanding context vs. query

### **2. Text Summarization**
A beginner project that generates concise summaries from longer text passages.

**Highlights**
- Demonstrates extractive or abstractive summarization techniques
- Uses NLP preprocessing methods to clean and structure text
- Shows how summarization models compress information
- Useful for news summarization, document reduction, and content recommendation

**Concepts Learned**
- Sentence ranking / keyword extraction  
- Sequence-to-sequence modeling fundamentals  
- Understanding how models detect important text  
- Evaluation using basic qualitative comparisons
