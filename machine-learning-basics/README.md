# Machine Learning Basics  
This directory contains beginner-level machine learning projects completed during my early learning phase.  
Each project uses simple, interpretable algorithms to understand core ML concepts such as data preprocessing, classification, model evaluation, and prediction on new inputs.

## üìÇ Projects Included

### **1. Customer Loan Prediction (Logistic Regression)**
A classification model that predicts whether a customer is likely to take a loan based on demographic and behavioral features.

**Highlights**
- Algorithm: Logistic Regression  
- Task: Binary classification (loan vs no-loan)  
- Accuracy: **89%**  
- Includes prediction on new customer inputs  
- Focuses on understanding coefficients, probabilities, and decision boundaries  

---

### **2. Email Spam Classification (Multinomial Naive Bayes)**
A text-classification project that identifies emails as **spam** or **ham**.

**Highlights**
- Algorithm: Multinomial Naive Bayes  
- Task: Spam detection  
- Accuracy: **97%**  
- Uses TF-IDF / Bag-of-Words text features  
- Includes prediction on new email text  

---

### **3. Lung Cancer Prediction (K-Nearest Neighbors)**
A simple ML model predicting lung cancer risk based on lifestyle and health indicators such as smoking, pollution exposure, genetic risk, and wheezing.

**Highlights**
- Algorithm: K-Nearest Neighbors (KNN)  
- Task: Health risk classification  
- Accuracy: **99.5%**  
- Good performance on new sample predictions  
- Helps understand distance metrics and feature scaling
